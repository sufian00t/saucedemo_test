# Playwright Test Automation for Swag Labs Application

## Prerequisites

- **Node.js** 16 or higher
- **npm** (included with Node.js)

## Quick Start

### 1. Install Playwright
```bash
npm init playwright@latest
```
### 2. Clone the Repository
```bash
git clone https://github.com/sufian00t/saucedemo_test.git
cd saucedemo_test
```
### 3. Run the script with Headed(GUI)
```bash
npx playwright test tests/saucedemo.spec.js --headed
```
### 4. Run the script without Headed
```bash
npx playwright test tests/saucedemo.spec.js
```
### 5. To Check HTML Test Report
```bash
npx playwright show-report
```
